🔐 What is GigVault?

GigVault is an Enterprise Zero Trust PKI Ecosystem built entirely in Go 1.23. It provides a complete Certificate Authority solution with modern security practices, Kubernetes-native deployment, and enterprise-grade features.

✨ Features

🔒

Custom Envelope Encryption

HSM-backed master key with database-stored encrypted DEKs. Memory-safe private key handling with automatic zeroing.

Zero Trust Architecture

JWT authentication (ECDSA P-256), RBAC authorization, mTLS inter-service communication, TLS 1.3 everywhere.

Production Ready

Kubernetes-native with Helm charts, PostgreSQL with SSL, comprehensive monitoring, and auto-scaling support.

🚀 Quick Start

Prerequisites

  • Go 1.23+
  • Docker
  • kubectl
  • Helm 3+
  • kind (Kubernetes in Docker)

Deploy Locally

# Clone the organization
git clone https://github.com/gigvault/infra
cd infra

# Create kind cluster
make kind-up

# Build all services
make build-all

# Deploy to local cluster
make deploy-local

# Run smoke tests
make smoke-test

Access Services

# CA Service
kubectl port-forward -n gigvault svc/ca 8080:8080

# Admin UI
kubectl port-forward -n gigvault svc/admin-ui 3000:3000

# View all services
kubectl get svc -n gigvault

  • Role-Based Access Control

  • Token bucket rate limiting
